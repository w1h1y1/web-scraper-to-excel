# Excel Data Cleaner Tool

A simple Python tool for cleaning Excel data automatically.

## Features
- Remove duplicate rows
- Remove rows with missing values
- Sort data by salary
- Export cleaned Excel file

## Tech Stack
- Python
- pandas
- openpyxl

## How to Run
1. Put the Excel file in the same folder as `main.py`
2. Name it `sample.xlsx`
3. Run:

```bash
python main.py
